JSCodeblocks

How to use:
1. Enter a JavaScript code you'd like to save.
2. Name it.
3. Save it.
4. Run it when you feel it.

Yes, this extensions's icon is a Hollow Knight reference.

Note: If the code input line is not empty, pressing 'Run' would run whatever's inside the input line. Please clear the input if you want to run a saved code block.
